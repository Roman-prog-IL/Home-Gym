## 🦾 Программа (Силовые)

### День А: Толкай (Грудь, Плечи, Трицепс + Ноги)
1. **Брусья (Грудь):** 3-4 подхода до отказа (цель 10-20 раз).
2. **Отжимания "Домиком" (Плечи):** 3 подхода (цель 8-15 раз).
3. **Болгарские сплит-приседания:** 3 подхода по 12-15 раз на каждую ногу (нога на диване).
4. **Алмазные отжимания (Трицепс):** 2-3 подхода (макс повторов).

### День Б: Тяни (Спина, Бицепс + Пресс)
1. **Подтягивания (Спина):** 4 подхода до максимума.
2. **Австралийские подтягивания:** 3 подхода (медленная фаза вниз!).
3. **Подтягивания обратным хватом (Бицепс):** 3 подхода.
4. **Подъем ног на турнике (Пресс):** 4 подхода.

---

## 🏃‍♂️ Беговая дорожка (Жиросжигание)

1. **LISS (Ходьба в гору):** Скорость 5.5-6.5 км/ч, **Наклон 6-8%**. 30-40 минут. (Не держаться за поручни!).
2. **HIIT (Интервалы):** 30 сек быстрый бег / 60 сек ходьба. Повторить 10 раз.
